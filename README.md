# splitflow-frontend

Welcome to the frontend repository for SplitFlow, a collaborative expense-sharing web application designed to simplify group finances with a clean, modern SaaS aesthetic.

---

## Tech Stack
- **Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (using the Soft Mint & Teal palette)
    - **Icons:** Heroicons
- **Backend-as-a-Service:** Supabase
- **State Management:** React Context API / Zustand (optional)
- **Icons:** Lucide React / Phosphor Icons
